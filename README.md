# PUKUL BAHLIL - Setup Instructions

## Struktur File
```
pukul-game/
├── index.html    ← Halaman utama
├── style.css     ← Styling (komik bold)
├── game.js       ← Logika game
├── img/
│   └── bahlil.jpg  ← TAMBAHKAN FOTO INI SENDIRI
└── README.md
```

## Cara Pakai

1. Buat folder `img/` di dalam folder `pukul-game/`
2. Taruh foto Bahlil dengan nama `bahlil.jpg` di dalam folder `img/`
3. Buka `index.html` di browser

## Fitur
- Klik foto atau tombol PUKUL! untuk memukul
- Upload foto sendiri (foto orang yang bikin kesal)
- Combo system: pukul cepat untuk combo multiplier
- Anger bar yang naik seiring pukulan
- FX kata-kata melayang (BAM, POW, HAJAR, dll)
- Screen shake saat combo tinggi
- Milestone pop-up di 10, 25, 50, 100, 250, 500 pukulan
- Shortcut keyboard: Space atau Enter = pukul
- Mobile-first, responsif di semua layar

## Catatan
Hanya untuk hiburan semata.
